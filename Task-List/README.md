# 🗒 Task List 

A simple web application to store your task 

### 🏁 Tech 
* html
* vanilla javaScript
* Materiallize for UI


### 🌟  Features!

  - Add a task to list
  - Delete any task items
  - filter your task list
  - task list is store in local storage . it means your task items persist after reloading the page
